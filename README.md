# Task-Management-System
Разработка и интеграция модулей программного проекта

## Инструкция по запуску проекта

Клонируем репозиторий:

```git clone https://github.com/LeVVeri/Task-Management-System.git```

Открываем папку проекта на компьютере.

Запускаем файл auth-module/login.html в браузере.

Радуемся жизни.
