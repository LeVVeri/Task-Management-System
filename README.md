# Task-Management-System
Разработка и интеграция модулей программного проекта

## Инструкция по запуску проекта

Клонировать репозиторий:

```git clone https://github.com/LeVVeri/Task-Management-System.git```

Открыть папку проекта на компьютере.

Запустить файл auth-module/login.html в браузере.

Радуемся жизни.
